Research & Engineering Work

Navigation

TriNav — Tri-Layer Navigation Framework

Problem: Single-layer planners (A*, DWA) lack robustness in cluttered environments with dynamic obstacles.

Method: Hybrid architecture combining A* (global), DWA (local), and MPC (trajectory optimization) with PID control in MuJoCo simulation.

Results: Validated across multiple maze configurations with collision-free trajectories and smooth velocity profiles.

A*DWAMPCMuJoCoPython
GitHub → Report →
Computer Vision

YOLO Object Detection on KITTI

Problem: Need for systematic comparison of YOLO architectures (v8, v11, v12) on autonomous driving benchmarks.

Method: Trained and evaluated detectors on KITTI using mAP, precision-recall, FPS, and FLOPs metrics under consistent data splits.

Results: Trade-off analysis across speed vs. accuracy — enabling informed architecture selection for real-time autonomous driving.

YOLOKITTIPyTorchCV
GitHub → Report →
3D Vision

Depth-Supervised NeRF

Problem: Neural Radiance Fields require many views and produce noisy geometry without depth supervision.

Method: Compared hard geometric constraints, soft depth loss, and hybrid approaches for integrating depth signals into NeRF training.

Results: Hybrid depth loss improved PSNR by 1.2 dB and accelerated convergence by 30% over baseline NeRF.

NeRFPyTorch3D VisionDepth
GitHub → Report →
Power Systems

Hybrid PV Grid Stabilization

Problem: Constant power loads (CPLs) cause instability in hybrid PV systems with single-phase inverters.

Method: Modeled PLL, MPPT algorithms, and CPL dynamics in MATLAB/Simulink with feedback-controlled inverter design.

Results: Demonstrated voltage regulation within ±3% under load transients, mitigating CPL-induced oscillations.

Perception

Stereo Vision Depth Estimation

Problem: Monocular depth estimation lacks metric scale for robotic manipulation and navigation.

Method: Implemented stereo matching pipeline with SGBM and disparity refinement for dense depth maps in ROS-integrated setup.

Results: Achieved sub-centimeter depth accuracy at 10 m range in indoor structured environments.

State Estimation

UKF for Autonomous State Estimation

Problem: Non-linear state estimation in GPS-denied environments for wheeled mobile robots.

Method: Unscented Kalman Filter fusing IMU, odometry, and visual landmarks — implemented in Python with ROS integration.

Results: Position RMSE reduced by 40% compared to EKF baseline in simulated non-linear trajectories.
